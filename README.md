## AirTeach
Это [Next.js](https://nextjs.org/) проект, созданный с помощью [`create-next-app@latest`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app) проект доступен по [ссылке на репозиторий](https://github.com/12mativ/air_food_front).

# Начало работы

Сначала склонируйте проект на Ваше устройство, выполнив в терминале команды:
```bash
# Спуститесь в нужную папку для установки проекта
cd <Имя папки>
# Склонируйте проект
git clone <Ссылка на репозиторий>
```
Разверните сервис приложения:
```bash
docker compose up
```

Запустите сервер для разработки:
```bash
npm run dev
# Или
yarn dev
```
Откройте [http://localhost:3001](http://localhost:3001) в своем браузере, чтобы увидеть результат.
